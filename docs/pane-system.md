# Pane System

Panes are the atomic working surfaces of the Operating Terminal.

## Core pane types
- Channel Pane
- Execution Pane
- Summary Pane
- Artifact Pane
- Portal Pane
- Telemetry Pane
- Oversight Pane

## Pane principles
- panes should be composable
- panes should be saveable in workspace layouts
- panes should have semantic identity
- panes should support drill-down from summary to raw execution
