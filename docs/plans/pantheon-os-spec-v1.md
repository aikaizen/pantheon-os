# Pantheon OS Spec v1

## Product definition
Pantheon OS is a human-sovereign, agent-native company operating system deployable for any company, with PromptEngines as the first internal deployment.

## Core surfaces
1. Principles
2. Company Dashboard
3. Operating Terminal
4. Runtime / Portal

## Principles
Defines sovereignty, org structure, agent framing, omnipotence within scope, omnidirectionality, and governance.

## Company Dashboard
The strategic surface that makes the company legible across ventures, operators, budgets, approvals, goals, and health.

## Operating Terminal
The tactical command-center IDE for live agent operations across chats, terminals, panes, tools, logs, and summaries.

## Runtime / Portal
The substrate beneath the visible surfaces: dual harnesses, portal state, supervision, event flow, telemetry, and memory.

## Immediate repo implications
- keep Pantheon OS framed as a general system
- keep PromptEngines framed as first deployment
- keep the dashboard as the most visible current surface
- grow the operating terminal as the next major surface
- expand registry, runtime, and pane/state models over time
