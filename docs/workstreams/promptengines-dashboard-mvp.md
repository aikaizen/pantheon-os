# Workstream A — PromptEngines Dashboard MVP

## Objective
Ship the fastest credible MVP of the PromptEngines dashboard.

## Repo target
This dashboard will be built in the PromptEngines repo once write access is provided there.

## Why it matters
The dashboard is the most visible part of the project and the clearest strategic surface.

## Scope
- portfolio visibility
- products / experiments / prototypes / apps
- approvals
- activity feed
- goals
- operators
- basic metrics surface

## Near-term outputs
- route map / IA
- MVP data contract
- live-ish portfolio view
- operator view
- approvals and activity view

## Success condition
A PromptEngines dashboard exists on the PromptEngines codebase and acts as the visible control-room surface for the company.
