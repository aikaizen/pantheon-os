
# Pantheon OS Skill Library

This is the first company skill library for Prompt Engines.

## Structure

- `company-os/` — operating system skills used across the whole company
- each skill package contains:
  - `SKILL.md`
  - `templates/`
  - `tests/`

## First 5 core company skills

1. Weekly Operating Review
2. Venture Status Refresh
3. Decision Gate Review
4. Goal Tree Planning
5. Build Stream Synthesis

## Standard

Each skill must:
- start from concrete use cases
- expose progressive disclosure
- be composable across agents
- define owner, version, status, review cadence, and venture scope
- be testable for triggering, functional success, and workflow improvement
