# PromptEngines Dashboard MVP

This is a migration-ready static PromptEngines dashboard built inside the Pantheon OS repo for speed.

## Why it exists here
- we want to move fast without waiting on the PromptEngines repo write token
- the dashboard can later be moved into the PromptEngines repo with minimal friction
- the folder is self-contained and static-host friendly

## Structure
- `index.html`
- `assets/styles.css`
- `assets/app.js`
- `data/dashboard.json`

## Current role
This is the strategic company dashboard surface for PromptEngines.
It is distinct from the Pantheon OS operating terminal, which will become the tactical command center.
