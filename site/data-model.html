← Pantheon OS

Cross-surface data model for PantheonOS as an observational and intervention surface over sovereign persona agents, channels, runtimes, controllers, and bindings. Persona agents remain canonical in agents.yaml, but they are no longer treated as the whole system.

◆Core object types

ObjectRegistryMeaning
Persona agentagents.yamlSovereign identity with soul, voice, memory, and mandate.
Runtimeruntimes.yamlExecution substrate where one or more persona agents live.
Controllercontrollers.yamlOverseer agent with host or machine access.
Channelchannels.yamlTelegram, CLI, or other conversational/control surface.
Bindingbindings.yamlRelationship between persona agents, runtimes, controllers, and channels.
